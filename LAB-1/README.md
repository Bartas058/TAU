# Shipment Tracker

Aplikacja REST API napisana w **Spring Boot 3 (Java 21)** umożliwiająca wyszukiwanie przesyłek po numerze śledzenia. Projekt zawiera warstwy: **Advice**, **Controller**, **Dto**, **Entity**, **Error**, **Exception**, **Repository**, **Service** i **Test**.

---

## Testy

W projekcie znajduje się około 10 testów jednostkowych, wykorzystujących różne asercje.

## Uruchomienoie

```bash
./gradlew test && ./gradlew bootRun
```
